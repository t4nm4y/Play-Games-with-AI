# Play Games with AI!
- A fully responsive website with a beautiful UI.
- Play various board games with AI or a friend.
- The AI uses a Mini-Max Algorithm with alpha-beta pruning. The difficulty of the AI can also be changed on the website.
#### It's hosted here: https://ai-games-tk.web.app/

## Tech Stack used:
React, Java-Script, HTML, CSS, Artificial-Intelligence
